<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these steps.

### Prerequisites

Install the latest version of npm.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Azurene/ever-after-bakery.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
